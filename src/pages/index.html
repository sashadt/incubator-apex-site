Apache Apex™

Enterprise-grade unified stream and batch processing engine.

Community Documentation Github

Enterprise Grade

Apex is a Hadoop YARN native platform that unifies stream and batch processing. It processes big data in-motion in a way that is highly scalable, highly performant, fault tolerant, stateful, secure, distributed, and easily operable.

Low Barrier-to-Entry

Write your business logic and leave all operability to the platform. It provides a simple API that enables developers to write or re-use generic Java code, thereby lowering the expertise needed to write big data applications.

Modular

The Apex platform comes with Malhar, a library of operators (modular units of business logic) that can be leveraged to quickly create new and non-trivial applications.
